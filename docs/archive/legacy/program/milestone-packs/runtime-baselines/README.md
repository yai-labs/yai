# Runtime Baselines (Program)

Program baseline packs organized by domain to avoid flat baseline blobs.

## Domain Sections
- `workspace/`
- `distributed-runtime/`
- `data-runtime/`
- `governance-runtime/`
- `operations-foundation/`

## Navigation Order
1. `operations-foundation/` (global baseline entry)
2. `workspace/`
3. `distributed-runtime/`
4. `data-runtime/`
5. `governance-runtime/`

## Rules
- Add new runtime baseline packs under one domain section only.
- Keep live operational procedure docs under `docs/runbooks/`.
- Use `docs/archive/` for superseded baseline artifacts.
