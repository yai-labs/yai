# Archive

Non-canonical documentation retained for history, migration traceability, or audit continuity.

## Areas

- `legacy/` superseded references retained for audit use.
- `migration/` migration notes, maps, and historical tranche artifacts.
- `historical-architecture/` obsolete architecture trees no longer canonical.

## Rules

- Archive content must not be referenced as live source-of-truth when a canonical doc exists.
- New canonical documentation must never be added in archive paths.
