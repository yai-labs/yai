# 25 Templates

Canonical templates for governance artifacts in this repository.

Use these templates when creating new documents:
- ADR template: `docs/20-governance/25-templates/adr/ADR-000-template.md`
- Proposal template: `docs/20-governance/25-templates/proposals/PRP-000-template.md`
- Runbook template: `docs/20-governance/25-templates/runbooks/RB-000-template.md`
- Milestone Pack template: `docs/20-governance/25-templates/milestone-packs/MP-000-template.md`

Rules:
- Keep IDs stable (`ADR-`, `PRP-`, `RB-`, `MP-`).
- Add explicit traceability links (proposal -> ADR -> runbook -> MP -> evidence).
- Do not publish closure artifacts with missing mandatory evidence.
