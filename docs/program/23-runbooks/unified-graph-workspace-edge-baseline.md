# Unified Graph Workspace + Edge Baseline (QG-2)

## Objective

Validate that owner runtime can project a unified graph summary spanning
workspace sovereignty and governed distributed runtime state.

## Baseline checks

1. Verify graph summary includes source/edge, mesh, delegation, transport, ingress, and overlay families.
2. Verify summary differentiates observed/accepted/canonicalized state layers.
3. Verify peer overlap/coverage markers are representable in unified graph context.
4. Verify transport/ingress signals are visible but not interpreted as authority.
5. Verify workspace graph remains owner-side adjudication anchor.

## Expected outcomes

- Unified graph summary is usable for operator and qualification narratives.
- Distributed runtime state is correlated owner-side without authority drift.
- Graph projection supports downstream CLI and AI grounding consumption.

## Anti-drift assertions

- Graph-linked != canonicalized by default.
- Overlay-connected != legitimacy-granted.
- Awareness-linked peers != distributed sovereign authority.
