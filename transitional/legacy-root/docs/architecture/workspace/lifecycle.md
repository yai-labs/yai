# Workspace Lifecycle

Canonical state and lifecycle model for workspace operation.

## Runtime Surfaces
- Workspace lifecycle APIs: `include/yai/runtime/`
- Lifecycle implementation: `lib/runtime/lifecycle/`
- Session/workspace state: `lib/runtime/session/`, `lib/runtime/workspace/`

## Covers
- state model and transitions
- lifecycle progression and recovery
- binding-state interactions
- inspection/read-model boundaries
