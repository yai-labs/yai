# yai-law Sunset Notice (Repository-Level)

`yai-law` is sunset as an operational dependency for `yai`.

Canonical home for governance runtime content is now:

- `yai/governance/`

Canonical docs and policy framing are now in:

- `yai/docs/`
- root policy files (`README.md`, `FOUNDATION.md`, `GOVERNANCE.md`, `COMPATIBILITY.md`, `VERSIONING.md`)

If an external `yai-law` repository is still visible, it should be treated as archived/migration-only and should redirect contributors to `yai`.
